# Other Commands & Things To Know

`/convert amount: 10 from: USD to: EUR`

Converts currency using three-letter currency codes.

`/poki`

Shows a random Pokemon.

`/version`

Shows the current Lerche version.

If Lerche does not respond to a TTS message, check:

- The message is in the configured TTS text channel.
- The user has `Lerche Listens` or `Amelia Listens`.
- The user is currently in a voice channel.
- Lerche has permission to connect and speak.
