# TTS - How To Use

To make Lerche speak:

1. Join a voice channel.
2. Go to the configured TTS text channel.
3. Type your message.
4. Lerche will join your voice channel and read it aloud.

Messages must be `400 characters` or less.

Mentions and links are cleaned up before being spoken, so Lerche will try to say them in a more natural way.
