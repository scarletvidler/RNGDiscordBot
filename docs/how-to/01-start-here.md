# [Start Here] How To Lerche.

Lerche is a Discord TTS bot. She reads messages from a chosen text channel and plays them aloud in the voice channel you are currently in.

To get started, your server needs:

- A text channel for TTS messages, usually `tts`.
- A text channel for bot updates, usually `lerche-updates`.
- A role named `Lerche Listens` or `Amelia Listens` for people allowed to use TTS.

Only members with `Lerche Listens` or `Amelia Listens` can send messages for Lerche to read.

Lerche also needs permission to view channels, send messages, read message history, connect to voice channels, and speak in voice channels.
