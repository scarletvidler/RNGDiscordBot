# TTS - Set Lerche's Voices Up

Lerche uses a default voice for most users.

If a user has a role named `male`, Lerche will use the configured male voice for them instead.

Admins can set the voice IDs with:

`/tts-female-voice voice-id: <ElevenLabs voice ID>`

`/tts-male-voice voice-id: <ElevenLabs voice ID>`

These voice IDs come from ElevenLabs.
